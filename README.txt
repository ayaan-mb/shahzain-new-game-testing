Premium-style original 3D driving world for agency service exploration.

== Installation ==
1. Create a ZIP of this plugin where `speedx-3d-drive.php` is at the ZIP root.
2. In WordPress, go to Plugins > Add New > Upload Plugin.
3. Upload the ZIP and activate plugin.
4. Add shortcode `[speedx_3d_drive]` to any page.
5. Configure plugin at **SpeedX 3D Drive** admin menu.
